# Get Started

## Instalation

Installing @helloinnovation/styled only takes a single command and you're ready to roll:

```cmd
# with npm
npm install --save @helloinnovation/styled

# with yarn
yarn add @helloinnovation/styled
```

## PostCSS & PostCSS Plugins

- **[PostCSS Nested](https://github.com/postcss/postcss-nested):** allows nesting like scss.
- **[PostCSS Prefix Selector](https://github.com/RadValentin/postcss-prefix-selector):** preffix every CSS selector with a custom namespace.
- **[Autoprefixer](https://github.com/postcss/autoprefixer):** plugin to parse CSS and add vendor prefixes to CSS rules using values from [Can I Use](https://caniuse.com/).

## VSCode Plugins

- **[PostCSS Language Support](https://marketplace.visualstudio.com/items?itemName=csstools.postcss)**: proper syntax highlighting.
